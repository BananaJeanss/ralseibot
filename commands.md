# ralseibot Command Reference

A quick reference for all slash commands supported by **ralseibot**.

---

## Utility Commands

| Command    | Description                                    |
| ---------- | ---------------------------------------------- |
| `/ping`    | Replies with Pong!                             |
| `/uptime`  | Replies with the bot's uptime                  |
| `/about`   | Replies with information about the bot         |
