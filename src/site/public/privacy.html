Privacy Policy

Last updated: July 16, 2025

1. Information We Collect

Automatically Collected Information:

  • Discord User ID (for command execution and cooldowns)
  • Server/Guild ID (for bot functionality)
  • Command usage data (for performance monitoring)

Information You Provide:

  • Text content when using text-based commands
  • Image URLs when using image manipulation commands

2. How We Use Information

We use collected information to:

  • Provide Bot functionality and execute commands
  • Implement rate limiting and cooldowns
  • Monitor performance and prevent abuse
  • Filter inappropriate content

3. Information Sharing

We do not sell, trade, or share your personal information with third parties, except:

  • When required by law
  • To protect our rights or safety
  • With your explicit consent

4. Data Storage and Security

Data is stored temporarily in memory for command execution and cooldowns. We do not maintain persistent databases of user data. Content filtering is performed locally.

5. Third-Party Content

The Bot aggregates content from public sources including Reddit and Twitter. These services have their own privacy policies.

6. Data Retention

User data is only retained temporarily during Bot operation and is not stored permanently.

7. Your Rights

You may stop using the Bot at any time. If you have concerns about your data, contact us through our Discord server.

8. Changes to Privacy Policy

We may update this policy. Continued use of the Bot constitutes acceptance of any changes.

9. Contact

For privacy questions, check the GitHub repository.
